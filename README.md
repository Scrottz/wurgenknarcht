# Project Wurgenknarcht

This repository contains the source code for a digital facade designed as a linguistic and psychological experiment. The objective is to analyze manipulation strategies, aggressive marketing, and internet scams by creating a perfectly calibrated "victim persona."

## Concept
The website `wurgenknarcht.at` presents itself as the private homepage of an elderly couple, Hans and Angela Wurgenknarcht.

The site is intentionally designed to be "perfectly imperfect." By employing targeted digital incompetence—such as distorted image aspect ratios, outdated color schemes, and optimization for Internet Explorer 6.0—it sends a specific signal to potential scammers: a technically overwhelmed but financially stable target.

## Technical Stack
Despite the visual appearance of a website from the 2010s, the underlying infrastructure is based on modern DevOps principles:

- **Frontend:** Static HTML5 & CSS3 (intentionally rigid design).
- **Version Control:** Git / GitHub.
- **CI/CD Pipeline:** GitHub Actions for automated deployment.
- **Deployment:** SSH/SCP transfer to a Linux-based web server (al-inkl).
- **Asset Generation:** AI-generated imagery (Gemini/DALL-E) to produce an authentic, amateur photographic style.

## Vibe-Coding and Satire
This project is approximately 95% "vibe-coded." This means that design decisions are not based on UX guidelines, but on the intuitive reproduction of "internet trash" and digital naivety.

**Purpose of the project:**
1. **Educational:** Implementation of a full CI/CD pipeline for static content.
2. **Satire and Analysis:** Dissecting the patterns of internet conspiracy theorists and marketers to examine their reaction patterns and logical fallacies within a controlled environment.

## Disclaimer
The characters "Hans and Angela Wurgenknarcht" are entirely fictional. The website is used exclusively for experimental and satirical purposes. No real personal data is collected, except within the framework of interacting with the (planned) lures to analyze spam patterns.
